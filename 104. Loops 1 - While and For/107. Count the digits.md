# Count the digits

## Problem Description
Take T (number of test cases) as input. </br>
For each test case, take integer N as input and Print the count of digits of that number.

***Note:*** </br> 
No of digits for number 0 is considered as 1.

### Problem Constraints (Important for test cases in interviews)
1 <= T <= 1000 </br>
0 <= N <= 100000000

---

### Input Format
The first line is the number T which denotes the total number of test cases. </br>
Next T lines contain an integer N for which you have to print the number of digits.

### Output Format
For T different Numbers, Print the number of digits in separate lines.

---

## Example Input
- **Input 1:** </br>
2 </br>
0 </br>
1

- **Input 2:** </br>
2 </br>
100 </br>
10101

  
## Example Output
- **Output 1:** </br>
1 </br>
1

- **Output 2:** </br>
3 </br>
5


## Example Explanation
- **Explanation 1:** </br>
0 and 1 both have only one digit.
  
- **Explanation 2:** </br>
100 has three digits and 10101 has 5 digits.

---

# CODE

```python
T = int(input())
for i in range(T):
    N = int(input())
    count = 0
    while N >= 0:
        count += 1
        N = N//10
    print(count)
```
