# Print perfect squares

## Problem Description
Take an input of a number A from the user. Print all perfect squares less than or equal to A. </br>
Note - Perfect squares are integers whose square root is an integer. (For Example: 16 is perfect square as âˆš16 = 4, or 42 = 16)

---

### Input Format
A single line consisting of a integer A.

### Output Format
Print perfect squares less than or equal to A in a single line in a space-separated manner.

---

## Example Input
- **Input 1:** </br>
20

- **Input 2:** </br>
100


## Example Output

- **Output 1:** </br>
1 4 9 16

- **Output 2:** </br>
1 4 9 16 25 36 49 64 81 100

---

# CODE

```python
A = int(input())
i = 1
# Start i from 1 and go until i*i is not greater than n
while i*i <= A:
    print(i*i, end=" ")
    i += 1
```

or 

### In function format

```python
def main():
    A = int(input())
    i = 1
    # Start i from 1 and go until i*i is not greater than n
    while i*i <= A:
        print(i*i, end=" ")
        i += 1
    return 0

if __name__ == '__main__':
    main()
```
