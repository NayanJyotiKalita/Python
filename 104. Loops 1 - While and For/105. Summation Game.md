# Summation Game

## Problem Description
Write a program to find sum all Natural numbers from 1 to N where you have to take N as input from user

### Problem Constraints
1 <= N <= 1000

---

### Input Format
A single line representing N

### Output Format
A single integer showing sum of all Natural numbers from 1 to N

---

## Example Input
- **Input 1:** </br>
5

- **Input 2:** </br>
10


## Example Output
- **Output 1:** </br>
15

- **Output 2:** </br>
55

---

# CODE

```python
N = int(input())
sum = 0
for x in list(range(1,N+1)):
    sum = sum + x

print(sum)
```
