# Print N stars

## Problem Description
Given an integer N, print N stars in a single line.

For example if N = 5 then pattern will be like:
```*****```

---

### Input Format
Single line input contains a single integer N.

### Output Format
Output N stars in a single line.

---

## Example Input
- **Input 1:** </br>
 2

- **Input 2:** </br>
 3


## Example Output
- **Output 1:** </br>
```**```

- **Output 2:** </br>
```***```


## Example Explanation
 Print the pattern as described.

---

 # CODE

 ```python
N = int(input())
for i in list(range(N)):
    print("*", end = "")
```

or

### In function format

```python
def printPattern(n):
    # Loop for N rows
    for i in range(n):
        print('*', end='')
        
def main():
    N = int(input())
    printPattern(N)

if __name__ == '__main__':
    main()
```

or

```python
def main():
    N = int(input())
    # Loop for N rows
    for i in range(n):
        print('*', end='')

if __name__ == '__main__':
    main()
```
