# Reverse Game

## Problem Description
Write a program that accepts T numbers(N) from the user and find reverse of the given number(N).

### Problem Constraints
1 <= T <= 100 </br>
1 <= N <= 100000000

---

### Input Format
First line is T which means number of test cases. </br>
Each next N lines contain an integer N.

### Output Format
T lines each containing reverse of the input integer.

---

## Example Input
- **Input 1:** </br>
2 </br>
101 </br>
105

- **Input 2:** </br>
1 </br>
100

## Example Output
- **Output 1:** </br>
101 </br>
501 </br>

- **Output 2:** </br>
1

## Example Explanation
- **Explanation 1:** </br>
Reverse(101)=101 </br>
Reverse(105)=501 

- **Explanation 2:** </br>
Reverse(100)=001=1

---

# CODE

### using for loop
```python
T = int(input())
for i in range(T):
    A = int(input())
    rev = 0
    while A > 0:
        last = A % 10
        rev = rev*10 + last
        A = A // 10
    print(rev)
```

### In function format:
```python
def main():
    tc = int(input())
    while tc > 0 :
        tc -= 1
        num = int(input())
        revs_number = int(0)  
        # Go from last digit to first digit and form new number
        while (num > 0):  
            revs_number = (revs_number * 10) + num % 10  
            num = num // 10
        print(revs_number)  
    return 0

if __name__ == '__main__':
    main()
```
