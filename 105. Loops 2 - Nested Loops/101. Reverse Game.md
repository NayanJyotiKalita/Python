# Reverse Game

## Problem Description
Write a program that accepts T numbers(N) from the user and find reverse of the given number(N).

### Problem Constraints
1 <= T <= 100 </br>
1 <= N <= 100000000

---

Input Format
First line is T which means number of test cases.
Each next N lines contain an integer N.

Output Format
T lines each containing reverse of the input integer.

---

Example Input
Input 1:
2
101
105

Input 2:
1
100

Example Output
Output 1:
101
501
Output 2:
1

Example Explanation
Explanation 1:
Reverse(101)=101
Reverse(105)=501
Explanation 2:
Reverse(100)=001=1
