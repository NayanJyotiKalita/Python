# Two Line Star Pattern

## Problem Description
Print a pattern consisting of N rows, where each row contains an asterisk (*) at the beginning and end of the line, with N-2 spaces in between.

The Pattern should look like: </br>
```*<N-2 Spaces>*```

Print the above pattern for a total of N Rows. </br>

Refer Example ouput, for better clarification. </br>


### Problem Constraints
2 <= N <= 100

---

## Input Format
First and only line of input contains a single integer N.

## Output Format
Output the pattern corresponding to the given N.

---

## Example Input
- **Input 1:** </br>
 2

- **Input 2:** </br>
 4

## Example Output
- **Output 1:** </br>
```
**
**
```

- **Output 2:** </br>
```
*  *
*  *
*  *
*  *
```

## Example Explanation
- **Explanation 1:** </br>
 Here N = 2,  So each row should have N - 2 spaces which is 0. </br>
 Thus the N rows(i.e, 2 rows) should be in the form (asterisk)(0 spaces)(asterisk)

- **Explanation 2:** </br>
 Here N = 4,  So each row should have N - 2 spaces which is 2. </br>
 Thus the N rows(i.e, 4 rows) should be in the form (asterisk)(2 spaces)(asterisk)

---

# CODE

```python
N = int(input())
for i in range(N):
    if N == 1:
        print("*")
    elif N == 2:
        print("**")
    else:
        for i in range(N):
            print("*" + " " * (N - 2) + "*")
```

'or'

```python
    N = int(input())
    # Loop for N rows
    for i in range(N):
        print("*" , end = "")

        print(" " * (N - 2), end = "")
```
