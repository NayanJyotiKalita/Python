# Stair Pattern

## Problem Description
Take an integer N as input, print the corresponding stair pattern for N.

For example if N = 4 then stair pattern will be like:
```
*
**
***
****
```

### Problem Constraints (Important for test cases in interviews)
1 <= N <= 100

---

### Input Format
First and only line of input contains a single integer N.

### Output Format
Output the stair pattern corresponding to the given N.

---

## Example Input
- **Input 1:** </br>
 2
 
- **Input 2:** </br>
 3

## Example Output
- **Output 1:**
```
 *
 **
```

- Output 2:
```
 *
 **
 ***
```

---

or

### In function format

```python
def main():
    N = int(input())
    # Loop for N rows
    for i in range(1,N+1):
        # Loop for i columns
        for j in range(1,i+1):
            print('*', end = '')
        print()    
    return 0

if __name__ == '__main__':
    main()
```
