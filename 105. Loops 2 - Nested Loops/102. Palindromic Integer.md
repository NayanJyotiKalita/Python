# Palindromic Integer

## Problem Description
You are given an integer A as input and you need to determine whether it is a palindrome or not. </br>
A palindrome integer is one whose digits, when reversed, result in the same number. </br>
For example, 121 is a palindrome because its reverse is also 121, but 123 is not a palindrome because its reverse is 321.

***Note:*** </br>
The given integer will not have any leading zeros. 

---

### Input Format
First and the only line contains a single integer A.

### Output Format
Print Yes if it is palindromic, else print No.

---

## Example Input
- **Input 1:** </br>
 120 
 
- **Input 2:** </br>
 1001 
 
- **Input 3:** </br>
 131 

## Example Output
- **Output 1:** </br>
 No 
 
- **Output 2:** </br>
 Yes 
 
- **Output 3:** </br>
 Yes 

## Example Explanation
- **Explanation 1:** </br>
 For A = 120, reverse(A) = reverse(120) = 021 = 21 (removing leading zeroes). 120 is not equal to 21 

- **Explanation 2:** </br>
 For A = 1001, reverse(A) = reverse(1001) = 1001, which is same as A.

- **Explanation 3:** </br>
 For A = 131, reverse(A) = reverse(131) = 131, which is same as A. 

---

# CODE

```python
def is_palindrome(A):
  original_num = A
  reversed_num = 0
  while A > 0:
    # Extract the last digit
    last_digit = A % 10
    
    # Reverse the number
    reversed_num = reversed_num * 10 + last_digit
    
    # Remove the last digit from the original number
    A //= 10
    
  # Check if the original and reversed numbers are equal
  if original_num == reversed_num:
    return "Yes"
  else:
    return "No"

num = int(input())
result = is_palindrome(num)
print(result)
```
