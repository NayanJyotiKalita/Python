# Calculate Interest

Considering the Interest() function:

```python
def Interest(p,c,t=2,r=0.09):
  return p*t*r
```

## Q. Which of the following function calls are legal?

**Choose the correct answer from below, please note that this question may have multiple correct answers**

    A. Interest(p=1000,c=5)

    B. Interest(r=0.05,5000,3)

    C. Interest(500,t=2,r=0.05)

    D. Interest(c=4,r=0.12,p=5000)

---

# Answer
