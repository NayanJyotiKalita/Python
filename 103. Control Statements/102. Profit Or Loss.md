# Profit Or Loss

## Problem Description
You are given the Cost Price C and Selling Price S of a Product. You have to tell whether there is a Profit or Loss. Also, calculate total profit or loss.

***NOTE:*** It is guaranteed that Cost Price and Selling Price are not equal.
***NOTE:*** You have to take input of the Cost Price(C) and Selling Price(S) from the user.

### Problem Constraints
1 <= C, S <= 109 </br>
C â‰  S

---

## Input Format
First line of the input contains a single integer C. </br>
Second line of the input contains a single integer S.

## Output Format
Print two integers in separate lines. </br>
First integer denotes whether there is a profit or loss. If there is a profit, print 1, else print -1. </br>
Second integer is a non-negative integer denoting the absolute value of total profit or loss.

---

## Example Input
- Input 1: </br>
 2 </br>
 4
- Input 2: </br>
 4 </br>
 1

## Example Output
- Output 1: </br>
 1 </br>
 2
- Output 2: </br>
 -1 </br>
 3

## Example Explanation
- Explanation 1: </br>
 Cost Price = 2 </br>
 Selling Price = 4 </br>
 As Cost Price < Selling Price, there is a profit. </br>
 Total Profit = Selling Price - Cost Price = 4 - 2 = 2

- Explanation 2: </br>
 Cost Price = 4 </br>
 Selling Price = 1 </br>
 As Cost Price > Selling Price, there is a loss. </br>
 Total Loss = Cost Price - Selling Price = 4 - 1 = 3 

---

# CODE

```python
C = int(input())
S = int(input())

if S > C:
    print(1)
    print(S - C)
else:
    print(-1)
    print(C - S)
```
