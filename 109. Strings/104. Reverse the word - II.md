# Reverse the word - II

Problem Description
You are given lowercase string (A) and you have to return after reversing that.

Problem Constraints
1 <= S.size() <= 1000


Input Format
First and only argument containing a lowercase string S.

Output Format
You have to return string which is reverse form of input string.


Example Input
Input 1:
interviewer
Input 2:
person

Example Output
Output 1:
reweivretni
Output 2:
nosrep


Example Explanation
Explanation 1:
Clearly, reverse of interviewbit is reweivretni.
Explanation 2:
Clearly, reverse of scaler is nosrep.

---

# CODE

```python
def solve(A):
    return A[::-1]
```
