# Simple Maths

## Problem Description
Perform the following operations on 6 and 3 and print their respective outputs in different lines.

1. Addition </br>
2. Subtraction </br>
3. Multiplication </br>
4. Division </br>

---

## Output Format
Print the output of 6+3, 6-3, 6*3, 6/3 in separate lines.

## Example Output
9 </br>
3 </br>
18 </br>
2 </br>

## Example Explanation
6 + 3 = 9 </br>
6 - 3 = 3 </br>
6 * 3 = 18 </br> 
6 / 3 = 2 </br>

---

# CODE

```python
print(6+3)
print(6-3)
print(6*3)
print(6/3)
```
